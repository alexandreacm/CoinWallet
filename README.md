## Running the project

Use **yarn** or **npm install** to install project dependencies. Then start the project.

```cl
yarn android or react-native run android/ios
```

## Running the tests

Use **yarn jest -u** or **yarn test** to execute the jest tests.

```cl
yarn jest or yarn test
```
